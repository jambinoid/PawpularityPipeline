# PawpularityPipeline

## Install

```
git clone git@github.com:jambinoid/PawpularityPipeline.git
poetry install
```
or
```
git clone git@github.com:jambinoid/PawpularityPipeline.git
python3 -m pip install .
```

## Activate shell
```
poetry shell
```

## Load data
```
python3 tools/load_dataset.py
```

## Train model
```
python3 train.py --config configs_examples/train.yaml
```
